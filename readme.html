# Template Wordpress Um repositório com um template inicial para iniciar novos projetos em Wordpress usando o pré-processador de estilos e scripts do laravel o Laravel Mix. ## Tecnologias - Sass - Bootstrap - Vue - Javascript - HTML - Laravel-Mix - PHP - Wordpress ## Como rodar o tema 1 - acesse a pasta do tema e rode `npm install` 2 - configure o arquivo `/wp-config.php` use o `/wp-config-sample.php` como base; 3 - crie um vhost na sua máquina 4 - acesse o vhost e instale o WP no banco de dados 5 - configure o arquivo `wp-content/themes/pasta-do-tema/webpack.mix.js` com o vhost 6 - acesse novamente `wp-content/themes/pasta-do-tema` e rode `npm run watch` ## Rodando o tema em modo de produção 1 - rode `git fetch --tags` para baixar as tags 2 - acesse a tag mais recente, por exemplo `git checkout tags/v0.0.1 -b tags/v0.0.1`
